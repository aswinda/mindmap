How to use the Mindmap

  • Enter: Create a sibling node
  • Ctrl + Enter: Create a child node
  • Click on a node: Select and edit
  • Double-click: Add/edit notes
  • Show Notes button: Toggle note visibility
  • Theme button: Switch between Modern and Roman themes
  • Delete key: Remove selected node
  • Arrow keys: Navigate between nodes
  • Left mouse drag: Move nodes around
  • Middle mouse drag: Pan the entire canvas
