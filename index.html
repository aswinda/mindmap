How to use the Neural Mindmap

  • Enter: Create a sibling node
  • Ctrl + Enter: Create a child node
  • Ctrl + N: Create a new root node
  • Ctrl + C: Toggle neural connection mode
  • Ctrl + D: Toggle remove connection mode
  • Click on a node: Select and edit (or connect in neural mode)
  • Click on input text: Edit node text
  • Double-click on node: Add/edit notes
  • Double-click on empty canvas: Create new node at cursor
  • Right-click on node: Customize node color and shape
  • Neural Connect Mode: Click two nodes to create neural connections
  • Remove Connection Mode: Click on connections to delete them
  • Delete key: Remove selected node (when not editing text)
  • Show Notes button: Toggle note visibility
  • Theme button: Switch between Modern and Roman themes
  • Arrow keys: Navigate between nodes
  • Left mouse drag: Move nodes around
  • Middle mouse drag: Pan the entire canvas
  • Alt + S: Create a new sticky note
  • Sticky Notes: Double-click to edit text, drag to move, resize from bottom-right corner
  • Sticky Settings: Click gear icon on sticky note to customize colors, fonts, and size
